# Collection SEO Content — Giant Hoodies

**Created:** 2026-02-17
**Covers:** Master Plan Tasks #31-35

## How to Use This Document

Open Shopify Admin → Collections. Work through each collection one at a time:

1. Click the collection
2. Scroll to the **Description** field → paste the **Visible Description**
3. Scroll to the bottom **Search engine listing** section → click "Edit"
4. Paste the **SEO Title** into the "Page title" field
5. Paste the **Meta Description** into the "Meta description" field
6. While you're there, check if the collection has a **Collection Image** — if not, add one
7. Save → move to the next collection

**Important — SEO title suffix:** The theme automatically appends ` – Giant Hoodies` to every page title (see `meta-tags.liquid` line 130). Do NOT add "Giant Hoodies" to the SEO titles below — it's already handled. The "Rendered Title" shown below is what Google will actually see.

**Priority order:** Collections are ordered by estimated traffic importance. Start at #1 and work down.

- **Character counts:** All SEO titles render under 60 characters with the auto-appended suffix. All meta descriptions target 145-160 characters.

- **Positioning:** No collection uses "wearable blanket" language per the copy edit guidelines. All copy uses "oversized hoodie" framing.
