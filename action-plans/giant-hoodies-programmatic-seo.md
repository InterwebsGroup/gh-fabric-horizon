# Programmatic SEO — Giant Hoodies

## The Opportunity

7,000+ designs already organized in collections. That's a massive data asset sitting on the site that probably isn't optimized for search. The play is turning the existing collection structure into search-optimized landing pages, then expanding with new keyword-targeted collections.

Three keyword patterns have real volume for this business:

**Pattern 1: `[theme] hoodie` / `[theme] oversized hoodie`**
- "cat hoodie", "tie dye hoodie", "dinosaur hoodie", "sunflower hoodie"
- These have real search volume. People search for specific designs.

**Pattern 2: `oversized hoodie for [persona]`**
- "oversized hoodie for women", "oversized hoodie for kids", "oversized hoodie for couples"
- Persona-based. Fewer pages but higher intent.

**Pattern 3: `gifts for [persona]` / `[theme] gifts`**
- "gifts for dog lovers", "gifts for nurses", "cozy gifts for her"
- Gift intent = ready to buy. Layered on top of design themes.

## Playbook 1: Themed Collections (Highest Priority)

**URL structure:** `/collections/[theme]-hoodies`

**Examples:**

| Collection | Target Keyword | Why |
|------------|---------------|-----|
| `/collections/cat-hoodies` | "cat hoodie" / "cat oversized hoodie" | Animal themes have high search volume |
| `/collections/dog-hoodies` | "dog hoodie" | Same — pet lovers search by animal |
| `/collections/tie-dye-hoodies` | "tie dye hoodie" | Popular style keyword |
| `/collections/dinosaur-hoodies` | "dinosaur hoodie" | Kids + adults search this |
| `/collections/sunflower-hoodies` | "sunflower hoodie" | Niche but converts |
| `/collections/halloween-hoodies` | "halloween hoodie" | Seasonal spike |
| `/collections/christmas-hoodies` | "christmas hoodie" | Massive seasonal volume |
| `/collections/anime-hoodies` | "anime hoodie" | Huge search volume in this niche |
| `/collections/sports-hoodies` | "sports hoodie" | Broad sports category |
| `/collections/funny-hoodies` | "funny hoodie" | High volume |
| `/collections/floral-hoodies` | "floral hoodie" | Style-driven |
| `/collections/patriotic-hoodies` | "patriotic hoodie" / "american flag hoodie" | Fits the Made in USA angle |

**How to build these on Shopify:**
1. Create automated collections using product title/type conditions (e.g., title contains "cat")
2. Each collection needs a **unique description** — not just "Browse our cat hoodies." Write 2-3 sentences about why this theme is popular, what makes your version special, who it's for.
3. Set a unique **SEO title** and **meta description** per collection
4. Add a collection image

**Template for collection descriptions:**

```
[Theme] Giant Hoodies — [Unique hook about this theme]. Browse [X]+
designs, all printed by hand in the USA. Intentionally oversized for
maximum cozy. Starting at $39 when you buy 3+.

[1-2 sentences specific to this theme — who loves it, when to wear it,
why it's popular as a gift]
```

**What makes this NOT thin content:**
- Each page has a genuinely different product set (different designs)
- Unique description per collection
- The products themselves are the unique value
- Real photos, real pricing, real reviews

**Estimated pages:** Start with 30, expand to 50-80 as you identify which themes have products AND search volume.

**URL structure:** `/collections/[occasion]-gifts` or `/collections/gifts-for-[persona]`

**Examples:**

|------------|---------------|--------|
| `/collections/christmas-gifts` | "cozy christmas gifts" | Seasonal (but keep live year-round) |
| `/collections/valentines-gifts` | "valentine gifts for her" / "valentine gifts for him" | Seasonal |

| `/collections/fathers-day-gifts` | "fathers day gifts" | Seasonal |
| `/collections/birthday-gifts` | "birthday gifts for her" | Evergreen |
| `/collections/college-gifts` | "college dorm essentials" / "going to college gifts" | Late summer spike |
| `/collections/get-well-gifts` | "get well soon gifts" | Evergreen |

**Why this works:** Gift searches have massive volume and extremely high purchase intent. The person searching is literally looking to spend money right now. An oversized hoodie is a natural gift — cozy, fun, one-size-fits-most removes the sizing anxiety that kills apparel gifting.

**Estimated pages:** 10-15.

## Implementation Plan

**Phase 1: Audit existing collections (Week 1)**
- List every collection currently live
- Check which ones have SEO titles/descriptions vs. blank
- Identify gaps — themes with products but no collection

**Phase 2: Optimize existing collections (Weeks 2-3)**
- Write unique SEO title + meta description for every existing collection
- Write a unique 2-3 sentence description for each
- Add collection images where missing

**Phase 3: Create themed collections (Weeks 3-5)**
- Start with the 10-15 highest-volume theme keywords
- Create automated collections based on product title/type matching
- Write unique descriptions for each
- Verify products are actually populating correctly

**Phase 4: Create persona + gift collections (Weeks 5-7)**
- Build persona collections — manually curate products for each
- Build gift collections — curate + write gift-angle descriptions
- Seasonal gift pages can go up anytime (Google indexes them for when demand spikes)

**Phase 5: Monitor + expand (Ongoing)**
- Check Google Search Console for impressions/clicks per collection page
- Double down on themes that get traction
- Add more niche collections as you spot keyword opportunities

├── /blogs/journal (content strategy posts)

**Key linking rules:**
- Every collection page links to the main `/collections/giant-hoodies` hub
- Related collections cross-link (cat-hoodies ↔ gifts-for-cat-lovers)
- Blog posts link to relevant collections (the "Best Oversized Hoodies" post links to themed collections)
- Product pages link to their themed collection ("More cat hoodies →")

## Quality Guardrails

- **Minimum products per collection:** Don't create a collection page with fewer than 6 products. Thin product pages get ignored by Google.
- **No duplicate collections:** "Cat Hoodies" and "Kitty Hoodies" targeting the same keyword = cannibalization. Pick one.
- **Unique descriptions required:** Never launch a collection with a blank or template-only description.
- **Noindex if thin:** If a collection has < 4 products temporarily, add `noindex` until it's filled out.

## What This Gets You

Conservatively: **60-120 new search-optimized pages**, each targeting a distinct keyword, built from products you already have. With a 5+ year domain and real press backlinks, these have a genuine shot at ranking — especially the long-tail themes where competition is low.

The best part: once the collections exist, they're self-maintaining. New products with matching titles/types auto-populate. You maintain them by keeping your product catalog organized, not by writing new content.
